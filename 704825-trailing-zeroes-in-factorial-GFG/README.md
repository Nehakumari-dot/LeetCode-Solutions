# [Trailing zeroes in factorial](https://www.geeksforgeeks.org/problems/trailing-zeroes-in-factorial5134/1)
## Easy
For an integer n, find the number of trailing zeroes in n!.
Examples :
Input: n = 5
Output: 1
Explanation: 5! = 120 so the number of trailing zero is 1.
Input: n = 4
Output: 0
Explanation: 4! = 24 so the number of trailing zero is 0.
Constraints:1 &lt;= n &lt;= 109
